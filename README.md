# Customer Churn Analysis using SQL & Python

## 📌 Project Overview

This project was developed as part of **Task 2 – SQL & Data Extraction** during the ApexPlanet Data Analytics Internship.

The objective was to build a structured SQL database from a cleaned customer churn dataset, write business-oriented SQL queries, and integrate SQL with Python for data extraction and analysis.

---

## 📂 Dataset

* IBM Telco Customer Churn Dataset
* Database: SQLite

---

## 🛠️ Technologies Used

* SQL
