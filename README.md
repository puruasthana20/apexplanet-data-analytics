# Task 3: Data Visualization & Dashboarding

## Objective

Develop interactive visualizations and executive dashboards to analyze customer churn patterns and support business decision-making.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Plotly
* Power BI

## Dashboard Features

* Executive KPI Cards

  * Total Customers
  * Total Revenue
  * Churn Rate
  * Average Monthly Charges
  * Average CLTV

* Interactive Visualizations

  * Churn by Contract
  * Churn by Internet Service
  * Payment Method Analysis
  * Monthly Charges Distribution
  * Customer Geographic Distribution
  * Customer Lifetime Value Analysis

* Interactive Filters

  * Contract
  * Internet Service
  * Gender
  * Senior Citizen
  * Payment Method

## Key Insights

* Customers with month-to-month contracts exhibited the highest churn rates.
* Customers with longer tenure demonstrated significantly higher retention.
* Fiber optic customers showed higher churn compared to DSL users.
* Higher monthly charges were associated with increased churn probability.
* Long-term contracts contributed to greater customer lifetime value.
